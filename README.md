# Quora CRUD App

A web application for creating, editing, and deleting posts with dynamic content management.

## Tech Stack
- Node.js
- Express.js
- EJS
- CSS

## Features
- Create posts
- Edit posts
- Delete posts
- Dynamic routing
- Responsive UI
- Glassmorphism design
